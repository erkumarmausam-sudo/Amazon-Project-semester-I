# Amazon & Flipkart Webpage Clone

This repository contains my first-year frontend web development projects based on Amazon and Flipkart webpage clones. The projects were created to practice and improve my skills in basic web development technologies.

## 🚀 Technologies Used

* HTML
* CSS
* React JS

## 📌 Project Features

* Responsive webpage layouts
* Navigation bars and homepage sections
* Product display cards
* Basic frontend styling and design
* Simple React component structure

## 🎯 Purpose of the Project

The main goal of this project was to learn and understand:

* Webpage structure using HTML
* Styling using CSS
* Basic React JS concepts
* Component-based frontend development
* UI design and layout building

## 💡 Learning Experience

Building these webpage clones helped me improve my frontend development skills, problem-solving ability, and understanding of modern web design concepts. It was a great hands-on learning experience during my first year.
